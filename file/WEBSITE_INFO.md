# AED Design & Serviços - Website Documentation

## Project Overview
This is the official website for **AED Design & Serviços**, a visual communication company based in Lichinga, Mozambique.
The website showcases the company's services, portfolio, and provides a contact channel for clients.

## Technical Stack
- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript
- **Deployment**: GitHub Pages (capable)

## Project Structure
- `/src`: Source code
  - `/components`: Reusable UI components (ServiceCard, Navbar, etc.)
  - `/pages`: Main page views (Home, Services, Portfolio, Contacts)
  - `/context`: Global state (Language)
  - `translations.ts`: All text content (PT/EN)
- `/public`: Static assets (images, logos)

## Key Features
1.  **Bilingual Support**: Full Portuguese and English translations.
2.  **Dynamic Content**:
    - **Services**: 12 detailed service categories.
    - **Portfolio**: 30+ project items with filtering.
    - **Products**: Popular products showcase.
3.  **Modern UI/UX**:
    - Responsive design for mobile and desktop.
    - Smooth scroll animations.
    - Lazy loading for performance.
    - Interactive "Glitch" text effects.
4.  **Contact Form**: Integrated with WhatsApp and email (UI only).

## Usage Commands
- `npm install`: Install dependencies.
- `npm run dev`: Start local development server.
- `npm run build`: Build for production.
- `npm run preview`: Preview production build.
